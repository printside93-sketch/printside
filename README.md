# Printside – Application de gestion d’imprimerie

Application complète pour gérer :
- Les ventes
- Les achats
- Les commandes
- Les clients
- Les fournisseurs
- Le stock
- Les impressions
- La caisse

## Structure du projet
- backend/  → API + base de données
- frontend/ → interface utilisateur
- database.sql → création des tables MySQL

## Fonctionnalités
- Gestion des commandes clients
- Gestion des produits et services (impression, photocopie…)
- Gestion des ventes et paiements
- Gestion des achats et fournisseurs
- Suivi du stock
- Tableau de bord
